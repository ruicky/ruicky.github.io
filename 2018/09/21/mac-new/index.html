新 MAC，如何快速上手？

发表于 2018-09-21 | 更新于: 2021-11-16 | 分类于 教程 |
字数统计: 674 | 阅读时长 ≈ 2


前言

最近收到新的 mac 电脑，说要配置一下环境，感觉猛的一下不知道，该如何入手。于是有了这篇文章，主要是帮大家，快速的搭建起属于自己的 MAC 环境。

正文

相信大家都已经开机，并已经完成基础的配置了，接下来就是一些大家常用的软件整理。

1. 更新系统

  • 打开 APP STORE
  • 在上面标签中选择 更新
  • 找到 软件更新 点击 更新 按钮

    PS:估计得1到2个小时，看网速。

2. 安装 沟通工具

  • 微信
    • 下载地址，点击跳转
    • macOS 版微信小助手 (PS:可防撤回，自动回复，多微信，免认证登录等)
  • QQ
    • 下载地址，点击跳转
  • 钉钉
    • 下载地址，点击跳转

3. 安装 ShadowsocksX-NG

  • 下载地址，点击跳转

  • Visual Studio Code：编辑器
    • 下载地址，点击跳转
  • Xcode：开发工具
    • 在 App Store 下载
  • Postman：接口调试工具
    • 说明：该工具是个基于谷歌浏览器的插件，所以请先科学上网，并安装了 Chrome
    • 下载地址，点击跳转
  • Docker：容器
    • 说明：该网站貌似也被墙了，所以你懂得。

  • Sourcetree：git图形化工具
    • 下载地址，点击跳转
  • Microsoft Remote Desktop：MAC 远程连接 Windows
    • 下载地址，点击跳转
  • iTerm2：命令行工具
    • 下载地址，点击跳转

  • Transmit：功能齐全的 FTP 客户端
    • 下载地址，点击跳转
  • Sketch：轻量易用的矢量设计工具
    • 下载地址，点击跳转
