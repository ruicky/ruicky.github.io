#SSH远程登录服务器

最近搭建了一个远程 Linux 服务器，在使用过程中每次SSH登录都需要输入密码，很不方便。所以想配置一下 SSH Key 实现 SSH 免密登录。同时，还要保证配置的过程中不能覆盖当前配置好的 SSH Key，比如 Github使用到的SSH。

所以现在有两个问题需要解决：

问题1： 登录时不用输入密码进行验证。
